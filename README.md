Program Details

MotorPH Basic Payroll System is a Java console application that processes employee payroll using attendance records from a CSV file. The system includes a login system for employees and payroll staff, allows employees to view their information, and enables payroll staff to process payroll for either one employee or all employees. The system calculates payroll based on cutoff periods and computes government deductions based on the monthly gross salary, which are applied only to the second cutoff.

How to Run the Program
	1.	Open the project in NetBeans.
	2.	Make sure motorph_payroll_data.csv is inside the project folder.
